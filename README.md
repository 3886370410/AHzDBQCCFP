# 图书管理系统小程序 java147

## 项目概述

图书管理系统小程序是一款集图书管理、借阅、归还等功能于一体的应用。该系统后端采用Java SSM框架，前端基于uniapp小程序开发。

## 技术栈

- 后端：Java, SSM, MyBatis, MySQL
- 前端：jQuery, HTML, Vue, uniapp

## 功能模块

### 小程序部分

- 登录
- 首页展示
- 系统公告
- 我的待还
- 图书归还
- 催还提醒
- 我的收藏管理
- 意见反馈

### 后台部分

- 首页
- 个人中心
- 用户管理
- 图书分类管理
- 图书信息管理
- 我的待还管理
- 图书归还管理
- 催还提醒管理
- 系统管理

## 系统角色

- 普通用户：登录后可借阅、归还图书，查看个人待还信息，管理收藏等。
- 管理员：负责用户管理、图书分类及信息管理、催还提醒管理等。

## 运行环境

- 后端：Java 1.8+，MySQL 5.7+
- 前端：uniapp小程序运行环境

## 部署步骤

1. 准备后端运行环境，安装Java和MySQL。
2. 导入数据库表结构及数据。
3. 部署后端代码到服务器。
4. 开发和构建前端uniapp代码。
5. 部署小程序到对应平台。
